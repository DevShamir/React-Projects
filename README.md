# React Projects Portfolio
This repository contains a collection of frontend applications and technical exercises developed to build proficiency in React. The projects range from foundational component exercises to responsive, multi-page web applications demonstrating state management, routing, and modern UI design.

## Featured Projects

### DevNotes (Blog Application)
A modern blog interface focused on readability and user experience.
* **Features:** Implements a light and dark theme toggle, dynamic routing for individual articles, reading progress indicators, and responsive grid layouts.
* **Technical Focus:** React Router integration and advanced CSS styling techniques.

### Glasses Store (E-Commerce UI)
A frontend prototype for an eyewear retail store.
* **Features:** Includes product category filtering, individual product cards, and a shopping cart interface with quantity adjustments and order subtotal calculations.
* **Technical Focus:** State management for cart operations and dynamic data rendering.

### DashApp (User Dashboard)
A user account management and settings interface.
* **Features:** Contains a sidebar navigation system, profile data forms, and a settings panel with interactive toggles for user preferences.
* **Technical Focus:** Form handling, separated state logic, and component-based architecture.

## Core Concepts and Exercises

Alongside the primary applications, this repository includes hands-on implementations of essential React concepts:
* **Props & State:** Passing data and managing local component memory.
* **Component Architecture:** Building reusable UI elements using static data.
* **Forms & Controlled Inputs:** Managing separated state for form elements.
* **Routing:** Implementing page navigation using React Router.

## Technology Stack
* React.js
* React Router v6
* HTML5 & CSS3 (Grid, Flexbox, Custom Variables)
